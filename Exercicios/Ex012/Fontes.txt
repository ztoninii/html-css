!Peso, estilo e shorthand!

CSS- 
body {
    font-family: Arial, Helvetica, sans-serif;
    /*
    Medidas Absolutas 
    cm, mm, in, px, pt, pc

    Medidas Relativas  
    em, ex, rem, vw, vc, %
    16px geralmente = 1em 
    */
    font-size: 1em;
    font-weight: bolder;
    font-style: italic;
    text-decoration: underline;

    /* shorthand font
    - font-style -> font-weight -> font-size -> font-family */
    font: italic bolder 3em Arial, Helvetica, sans-serif;

    /* lighter normal bold bolder */
    /* weight - peso
       width - largura 
       height - altura 
    */
}


Como usar fontes externas 


@font-face {
    font-family: "love";
    src: url("fonts/love larry ttf.ttf") format("truetype");
    font-weight: normal;
    font-style: normal;
   /*
        tipos de format()
        - opentype (otf)
        - truetype (ttf)
        - embedded-opentype
        - truetype-att (Apple Advanced)
        - svg 
    */

 h1{
        font-family: "love", Times, serif
        font-size: 3em;
        font-weight: normal;
    }
